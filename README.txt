# SmartLINK - AI-Powered Recommendation Engine

SmartLINK is an AI-based recommendation engine that connects people using shared institutions (e.g., schools, companies), activities, and social graphs. It utilizes **Qdrant** for semantic search and **Neo4j** for graph database management.
